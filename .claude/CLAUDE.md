# Autonomous Marketing Agent — Claude Code SDK Context

## What this project is
Autonomous Marketing Agent is an autonomous AI marketing platform. It runs weekly intelligence pipelines, generates ad creatives, auto-launches Meta Ads campaigns, monitors performance, and improves itself over time.

## Stack
- NestJS + TypeScript (backend framework)
- MongoDB + Mongoose (data layer)
- BullMQ + Redis (job queues)
- Claude Code SDK `query()` (all AI agent calls)
- Meta Ads MCP (campaign management)
- n8n (delivery to Slack/WhatsApp/Email/Notion)

## Critical Rules
1. NEVER hardcode product names, prices, or dates in agent system prompts
2. ALL budget/safety checks are in TypeScript — Claude agents cannot override them
3. EVERY database query must include a `tenantId` filter
4. EVERY S3 path must be prefixed with `tenantId/`
5. EVERY agent call goes through `ClaudeService.runAgent()` — never call `query()` directly

## Agent System Prompt Architecture
- Strategic prompts stored in MongoDB per company (`company.prompts.*`)
- Live data (products, prices, promotions) injected at runtime via `LiveContextBuilder`
- Prompts are regenerated by `PromptGeneratorService` when brand-relevant fields change
